# Smart Contact Manager (C)

A C application that manages contact information using a singly linked list. The project allows users to add, search, delete, sort, and store contact details in a file for persistent storage.

## Features

- Add new contacts
- Search contacts by name
- Delete existing contacts
- Sort contacts alphabetically
- Save contacts to file
- Load contacts from file

## Project Structure

The application separates logic into multiple modules including contact management functions, file handling operations, and the main driver program. This structure keeps the code organized, modular, and easy to maintain.
